### @nestjs/throttler

### slowapi

## Security standards

### OWASP Top 10 (web)
Current version: 2025 ranking. Supply chain attacks rose to A03:2025. A10:2025 is Mishandling Exceptional Conditions. No category removals from 2021 ‚Äî new ordering reflects threat landscape shift.

### OWASP LLM Top 10
Current version: v2.0. Covers prompt injection (LLM01), sensitive disclosure (LLM02), supply chain (LLM03), improper output handling (LLM05), excessive agency (LLM06), vector/embedding weaknesses (LLM08), unbounded consumption (LLM10).

### OWASP Top 10 for Agentic Applications
Current version: 2026 (first edition). Covers multi-agent orchestration risks: privilege escalation across agent boundaries, plan hijacking, unsafe memory persistence. Complements (does not replace) OWASP LLM Top 10 ‚Äî LLM Top 10 covers model-layer risks; Agentic Top 10 covers orchestration-layer risks.

### NIST SP 800-63B
Current guidance: password minimum ‚â•12 characters (authenticator assurance). No SMS OTP for AAL2+ ‚Äî TOTP or hardware key required. Password hashing: argon2id recommended.

### AWS Responsible AI Lens
Current version: re:Invent 2025 edition. 10 dimensions: Controllability, Privacy, Security, Safety, Veracity, Robustness, Fairness, Explainability, Transparency, Governance. Use alongside OWASP LLM Top 10 for pre-launch AI reviews.

### AWS AIDLC
Current guidance covers: prompt injection mitigations at input boundaries, structured output validation (treat model output as untrusted external API), agent trust boundaries (explicit tool allowlists, max-steps guards, human-in-the-loop for irreversible actions), data minimization before LLM context injection.
